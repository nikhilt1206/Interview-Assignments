Interview Assignments

This repository contains a collection of Java-based interview assignments and practice problems.
It focuses on strengthening core programming concepts, problem-solving skills, and basic data handling techniques commonly asked in technical interviews.

📌 Topics Covered

* Core Java
    * Object-Oriented Programming
    * File Handling (Serialization & Deserialization)
* JSON Processing
    * Gson (Serialization & Deserialization)
    * Jackson (Serialization & Deserialization)
* Problem Solving
    * String manipulation problems
    * Array-based problems
    * Logical coding exercises
* POJO Design
    * Custom object modeling using Java classes
 
📂 Project Structure
    * src/main/java/DeutscheBankAG/

Contains all assignment-related classes including:

* Serialization & Deserialization examples
* JSON handling using Gson & Jackson
* String and array problem solutions
* Utility and POJO classes

▶️ How to Run

1. Open the project in IntelliJ IDEA (or any Java IDE)
2. Ensure JDK 21 (or compatible version) is configured
3. Run any class containing a main() method

Example:
Deserialization_Runner.java
ReverseEachWordOfStringTwoPointer.java
